кожна папка містить: 
- скрипт побудови графіків кожної країни для питання 1 та 2
- hotelBookings - зображає кількість бронювань відповідного типу готелю для країни
		  та вілсоток бронювань певно типу готелю серед усієї кількості броні країни) 
- month_hotelBookings - зображає вибір типу готелю в залежності від місяця

файл Аll_Coutrys відфільтровує топ-15 країн та будує загальний графік для питання 1
coutries_hotelType_bookings_percentage - зображає результат для 15 країн як у hotelBookings
